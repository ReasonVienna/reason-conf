## Develop

Install all depedencies via

```sh
yarn install
```

Run BuckleScript in one Shell

```sh
yarn re:start
```

Run Gatsby in another Shell

```sh
yarn start
```

Visit `http://localhost:8000` in your browser

## Build

```sh
yarn re:build && yarn build
```
